¿Que es angular?

Angular es un Framework (Un marco de trabajo) para la creación de aplicaciones web de una sola página o también conocidas como SPA. Angular separa totalmente el Fronted y el Backend de una aplicación, evita el código repetitivo y esta basado en la arquitectura MVC. Angular fue creado por Google y continúa añadiendo nuevas características en cada nueve versión. Angular esta totalmente integrado con TypeScript como lenguaje de programación.

El CLI de Angular

El CLI de angular es una interfaz que permite al desarrollador a crear partes de nuestra aplicacion, como modulos, componentes, directivas, etc., en cuestion de segundos. Tambien ayuda a la creación inicial de nuestro proyecto y genera archivos y carpetas debidamente organizados.

Para usar el CLI de Angular deberemos instalar Angular, y como prerequisito debemos tener instalado NodeJS en el equipos.

Elementos basicos de Angular

Angular nos provee de muchos elementos para la creación de nuestra pagina web, entre ellos tenemos:

  • Modulos
  • Componentes
  • Directivas
  • Pipes

Estos no son los únicos elementos que nos provee angular, pues, tengamos en cuenta que Angular es un Framework potente para el desarrollo web, pero teniendo conocimientos de estos elementos básicos, podremos crear nuestra primera aplicación totalmente funcional.

Módulos

Los módulos al igual que cualquier otro elemento en Angular son clases que tienen una estructura especifica que hay que respetar para que angular lo trate como tal.

Los módulos no se apartan de este concepto, son clases que poseen un decorador que convierte una sencilla clase en un módulo, componente, directiva o pipe. Los módulos pueden ser vistos como almacenes donde podremos tener a disposición los componentes, servicios o incluso otros módulos que nuestra aplicación web necesite a lo largo de su desarrollo. Una aplicación de Angular depende de un modulo principal que es llamado Modulo Raíz y del cual pueden enlazarse otros módulos que contengan sus propios componentes y servicios. Se considera buena práctica dividir una aplicación web de Angular en distintos Módulos y así armar un árbol jerárquico de módulos considerando al modulo principal como nodo principal del árbol.

La forma de crear un Modulo en Angular es usando el cli de angular:

>>> ng generate module nombre del modulo

Componentes

Los componentes son los bloques más básicos de angular, son aquellos que se encargan de mostrar la parte visual de la aplicación web y de proporcionar funcionalidad (lógica) a su correspondiente elemento visual. En pocas palabras un componente consta primordialmente de lo siguiente:

  • Un template HTML (Vista)
  • Un archivo de lógica TypeScript (Controlador)

Opcionalmente un componente puede enlazar uno o varios archivos CSS o SCSS para dar estilos adicionales al template.

Al igual que un módulo, un componente es una clase de typescript que lleva un decorador Component, un componente en angular se puede crear mediante el CLI de angular como sigue:

>>> ng generate component nombre del componente

En primer lugar, tenemos las directivas estructurales que son directivas a nivel de etiquetas HTML, estas directivas nos permiten añadir, eliminar o manipular elementos del DOM. Estas directivas son fácilmente reconocibles ya que tienen como prefijo el símbolo (*). Algunas de las directivas estructurales que nos provee angular son:
  • ngIf
  • ngFor

Angular nos provee de algunos pipes prefabricados, a continuación, algunos de ellos:

  • async
  • json
